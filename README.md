## 🎯 **About The Project**

**Cyber Scam Detector** is a web-based tool that helps Indian citizens identify and prevent cyber scams. Users can paste suspicious messages, and the tool will analyze them for scam indicators, providing a risk score and detailed reasons.

### **Why This Matters:**
- 📱 **Every day in India:** People lose money through fake UPI requests
- 🎓 **Students:** Get trapped by fake internships and scholarships
- 💰 **Citizens:** Receive scam SMS and phishing attempts
- 🔐 **Most people:** Don't know how to identify these attacks

---

## 🛡️ **Features**

| Feature | Description |
|---------|-------------|
| 🏦 **Bank Fraud Detection** | Detects fake bank messages and phishing attempts |
| 🎓 **Scholarship Scam Detection** | Identifies fake scholarship and registration scams |
| 🎁 **Prize/Lottery Scam Detection** | Catches "You won" and lottery scams |
| 🔐 **OTP Scam Detection** | Detects OTP-related fraud attempts |
| 🆔 **Aadhaar Scam Detection** | Identifies Aadhaar-related scams |
| 📊 **Risk Scoring** | Shows risk percentage (0-100%) |
| 🔍 **Pattern Recognition** | Detects multiple scam patterns |
| 📋 **Copy Results** | Share detection results with others |
| 🚨 **Report Feature** | Direct link to cybercrime.gov.in |
| ⚡ **Quick Examples** | One-click testing with example scams |
| 📱 **Mobile Friendly** | Works on all devices |
